# cam

シリアルカメラを繋いだマイコンで写真を撮影し、WiFiを通じMQTT brokerに画像を飛ばすやつ
ポンプの制御機能を追加し、水耕栽培機能をもたせたい。
シリアルカメラから撮影した画像をタイムラプス映像にし、植物の成長を見守りたい。
